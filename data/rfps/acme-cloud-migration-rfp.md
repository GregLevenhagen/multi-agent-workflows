## 1. Company Overview

Acme Corporation is a Fortune 500 manufacturer of precision industrial equipment, serving clients in aerospace, automotive, and energy sectors for over 45 years. With $3.2B in annual revenue and 8,500 employees across 12 facilities, we are undertaking a strategic digital transformation initiative to modernize our technology infrastructure and accelerate innovation.

## 2. Project Background

Our enterprise currently operates a suite of legacy .NET Framework 4.8 applications supporting manufacturing execution, supply chain management, and customer relationship management. These applications run on aging on-premises Windows Server infrastructure with SQL Server databases. We have committed to migrating our core business applications to Microsoft Azure as part of our 2025-2026 technology modernization roadmap.

## 3. Engagement Type

**Staff Augmentation** — We require experienced consultants to embed within our existing IT team and work alongside our internal developers throughout the migration. Our internal team of 12 developers will continue to own the codebase; we need specialized Azure expertise to guide and accelerate the effort.

## 4. Scope of Work

### 4.1 Application Assessment & Migration Planning
- Inventory and assess 14 existing .NET Framework applications for cloud readiness
- Develop prioritized migration roadmap with dependency mapping
- Recommend migration strategies per application (rehost, refactor, rearchitect)

### 4.2 Infrastructure & DevOps
- Design and implement Azure landing zone using Azure Well-Architected Framework
- Establish CI/CD pipelines using Azure DevOps or GitHub Actions
- Configure Azure Kubernetes Service (AKS) clusters for containerized workloads
- Implement infrastructure as code using Bicep or Terraform

### 4.3 Application Migration
- Migrate legacy .NET Framework applications to .NET 8/9
- Migrate on-premises SQL Server databases to Azure Cosmos DB where appropriate, Azure SQL for relational workloads
- Implement Azure Service Bus for inter-application messaging
- Ensure zero-downtime migration with rollback capabilities

### 4.4 Security & Compliance
- Implement Azure AD (Entra ID) integration for identity management
- Configure network security groups, private endpoints, and Azure Firewall
- Ensure SOC 2 Type II compliance throughout migration
